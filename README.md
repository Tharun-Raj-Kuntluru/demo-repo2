# CampusNet Planner

A comprehensive web application for streamlining network design requests from educational institutions. This platform provides detailed project plans and real-time status updates through a multi-user system with role-based access.

## 🎯 Core Purpose

CampusNet Planner streamlines the network design workflow by connecting IT Directors from educational institutions with expert network architects, while providing administrators with efficient project management tools.

## 👥 Target Audience

### Three Distinct User Roles:

1. **Clients (IT Directors)**: Submit design requests and track project progress
2. **Administrators**: Manage workflow by assigning projects to architects
3. **Network Architects**: Design networks and update project statuses

## ✨ Key Features

- **Professional Landing Page**: Clean, modern design with role-based login
- **Client Portal**: Submit requests, monitor status, view dynamic network plans
- **Administrator Dashboard**: Review pending projects and assign architects
- **Network Architect Dashboard**: Access project details and design tools
- **Real-time Status Updates**: Live progress tracking and notifications
- **Comprehensive Request Form**: Detailed requirements submission
- **Project Management**: End-to-end workflow management

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with modern design principles
- **Icons**: Font Awesome 6.0
- **Responsive Design**: Mobile-first approach
- **Storage**: Local Storage for demo authentication

### Quick Start

1. **Access the Application**: Open `index.html` in your browser
2. **Login**: Click "Login" and select your role, then use "Demo Login" buttons
3. **Explore Features**: Each role has a different dashboard with role-specific features
4. **Submit Requests**: As a client, use "New Design Request" to submit projects
5. **Manage Projects**: As an admin, assign projects to architects
6. **Design Networks**: As an architect, access design tools and update project status
